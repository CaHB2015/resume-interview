# 更新日志

## [1.0.0] — 2026-06-24

### 新增

- 🎉 初始发布
- 完整的 10 步工作流：从代码分析到简历生成、面试准备
- 两种执行模式：快速模式（面试急用）和完整模式（深度准备）

### 核心功能

- **项目经历生成**：4-6 条简历用 bullet，每条包含业务场景 + 技术方案 + 具体动作 + 量化数据
- **项目自我介绍**：30 秒 / 1 分钟 / 3 分钟三个版本
- **面试题库**：覆盖架构、缓存、数据库、安全、并发、MQ、系统设计等方向
- **追问树**：模拟面试官追问逻辑链，帮助深度准备
- **升级路线图**：对比 JD 要求，找出差距并排序优先级

### 参考知识库

- **capability-map.md**：Java / Python / Go / 前端 技术栈到面试能力的映射
- **interview-topics.md**：Redis / MySQL / Spring Boot / JWT / RabbitMQ 等面试话题
- **jd-keywords.md**：6 种开发岗位（后端/前端/全栈/AI/数据/DevOps）的 JD 高频关键词
- **output-templates.md**：简历项目经历、自我介绍、面试题库等输出模板和好/坏示例
