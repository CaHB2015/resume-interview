## ATS Keyword Tips

### Action Verbs (use in project experience)
- 负责 (responsible for)
- 设计 (design)
- 实现 (implement)
- 优化 (optimize)
- 重构 (refactor)
- 搭建 (build)
- 引入 (introduce)
- 建立 (establish)
- 提升 (improve)
- 降低 (reduce)

### Technical Depth Indicators
- "基于...设计" (design based on...)
- "引入...机制" (introduce...mechanism)
- "构建...架构" (build...architecture)
- "通过...优化" (optimize through...)
- "解决...问题" (solve...problem)

### Metrics to Include
- Response time (P99 latency)
- Throughput (QPS / TPS)
- Availability (99.9% / 99.99%)
- Error rate reduction
- Performance improvement percentage
- Data volume handled
- Concurrent user support
