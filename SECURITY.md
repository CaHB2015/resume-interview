## 报告安全漏洞

如果你发现了安全漏洞，**请不要公开提交 Issue**。请通过以下方式私下报告：

- 通过 GitHub 的 Security Advisory 功能

我们会在 **48 小时内**确认收到报告，并在修复后公开披露。

## 安全最佳实践

本 Skill 是 Codex/Claude 的 Prompt + 知识库项目，不执行可执行代码。但请注意：

- `outputs/` 目录下的生成文件包含用户个人信息，已通过 `.gitignore` 排除
- 请勿在 Issue 或 PR 中分享他人的简历或面试内容
- 参考知识库文件中的示例均为虚构数据，不得直接使用

感谢你帮助维护社区安全！
